A2Z WorkHub — Sequence Call Diagrams

Sơ đồ trình tự cuộc gọi (Sequence Diagram) cho các luồng nghiệp vụ chính

1. Authentication & Session Flow

Luồng đăng nhập, xác thực JWT, và quản lý session. Bao gồm cả trường hợp refresh token khi session hết hạn.

Synchronous Call
Response / Return
Async Event
Error / Redirect

2. Task Creation Flow

Luồng tạo công việc mới: từ form nhập liệu → validation → tính KPI score → lưu DB → notification → realtime broadcast cho các user khác.

3. Revenue Management Flow

Luồng quản lý doanh thu: tạo entry (draft) → xác nhận (confirmed) → điều chỉnh (adjusted) → báo cáo tổng hợp. Bao gồm cả luồng cron tự động.

4. KPI Calculation & Bonus Allocation

Luồng tính KPI tự động hàng ngày qua Cron → Edge Function → tính điểm trọng số → phân bổ thưởng → phê duyệt → chi trả.

5. Contract & Budget Allocation Flow

Tạo hợp đồng → gán billing milestones → phân bổ ngân sách cho phòng ban → theo dõi nghiệm thu → liên kết doanh thu.

6. Realtime Synchronization Flow

Khi một user thay đổi data → PostgreSQL trigger → Supabase Realtime broadcast → Tất cả client đang subscribe nhận update → React Query auto-invalidate.

7. Report Generation & Export Flow

Tạo báo cáo: chọn loại report → API aggregates data → tính toán → render UI hoặc export file (Excel/Word/PPTX).
